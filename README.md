# vimrc

## Installation

1. clone repo

```
git clone https://github.com/undus5/vimrc.git ~/.vimrcs
```

2. link `~/.vimrcs/vimrc` to:

```
# Linux

~/.config/nvim/init.vim    # Neovim
~/.vimrc                   # Vim

# Windows

~/AppData/Local/nvim/init.vim    # Neovim
~/_vimrc                         # Vim
```
